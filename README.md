#  Jornada de Aprendizado em Programação

# Sobre
Este repositório documenta minha jornada de aprendizado em Ciência da Computação, com foco em:

- Python
- Banco de Dados (SQL)
- Lógica de Programação

O objetivo é consolidar fundamentos através de estudos práticos, exercícios e pequenos projetos.



#  Objetivos

- Desenvolver uma lógica de programação sólida
- Aprender boas práticas de código
- Criar o hábito de estudar e registrar evolução
- Construir uma base forte para oportunidades na área de tecnologia



#  Tecnologias e Conteúdos

-  Python (foco atual)
-  Banco de Dados (SQL)
-  Linguagem C
-  Lógica de Programação
-  Estruturas de Dados
-  Exercícios práticos
-  Pequenos projetos



#  Organização

O conteúdo está organizado por linguagem e tema, começando por Python e evoluindo para banco de dados e outros conceitos.



#  Observação

Este repositório faz parte do meu processo contínuo de aprendizado e evolução na área de tecnologia.
